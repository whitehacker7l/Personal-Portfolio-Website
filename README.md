# Personal Portfolio Website

This is my personal portfolio website built using HTML and CSS. It showcases my skills, projects, education, and contact details in a clean, structured, and responsive design.


# Features
- Fully responsive design (mobile, tablet, desktop)
- Modern dark theme UI
- Smooth hover animations and effects
- Hero section with introduction and image
- Skills showcase section
- Project showcase section
- Education section
- Contact form UI

# Tech Stack
- HTML5
- CSS3


# How to Run
1. Download or clone this repository  
2. Open `index.html` in your browser  
3. Done
